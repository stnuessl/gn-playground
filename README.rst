=============
gn-playground
=============

A repository to learn `gn <https://gn.googlesource.com/gn>`_.

.. contents::

Features
========

#. Include check
#. Metadata usage, gathering and enforcement
#. Use of different toolchains
#. (extendable) templates for binaries, components and unit tests
#. python action and utility scripts
#. component-based build example
#. compile_commands.json suiteable for use by a language server
